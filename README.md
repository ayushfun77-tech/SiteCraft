🚀 SiteCraft

SiteCraft is a professional web development service platform where clients can request custom websites, send messages, and book appointments.

It is designed to help individuals, businesses, and creators get high-quality websites built quickly and efficiently.

🌐 About the Project

SiteCraft allows users to easily connect with a developer (you 👨‍💻) to get their website built.

Instead of building websites themselves, users can:

📩 Send project requests
📅 Book appointments
💬 Communicate their requirements
🌍 Get fully customized professional websites
✨ Features
📩 Contact / Request Form
📅 Appointment Booking System
🎨 Clean and modern UI
